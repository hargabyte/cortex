# Code of Conduct

## Our Standards

This project is committed to providing a welcoming environment. We expect:

- **Be respectful**: Treat everyone with respect. Disagreement is fine; personal attacks are not.
- **Be constructive**: Offer helpful feedback. Criticism should be about code, not people.
- **Be patient**: Remember that everyone has different skill levels and backgrounds.

## Unacceptable Behavior

- Harassment, trolling, or personal attacks
- Discriminatory language or behavior
- Publishing others' private information
- Sustained disruption of discussions

## Enforcement

Violations may result in:
1. Warning
2. Temporary ban
3. Permanent ban

Report issues to hargabyte@gmail.com.
